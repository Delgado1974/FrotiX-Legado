## Visão Geral

A interface `IRecursoRepository` define o contrato específico para operações de repositório de recursos do sistema.

**Principais características:**

✅ **Herança**: Estende `IRepository<Recurso>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

**Descrição**: Retorna lista de recursos para DropDownList

---

### `Update(Recurso recurso)`

**Descrição**: Atualiza recurso com lógica específica
