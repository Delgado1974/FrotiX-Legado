# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `IMotoristaRepository` define o contrato específico para operações de repositório de motoristas.

**Principais características:**

✅ **Herança**: Estende `IRepository<Motorista>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetMotoristaListForDropDown()`

**Descrição**: Retorna lista de motoristas para DropDownList

---

### `Update(Motorista motorista)`

**Descrição**: Atualiza motorista com lógica específica
