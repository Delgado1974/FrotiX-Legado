# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `INotaFiscalRepository` define o contrato específico para operações de repositório de notas fiscais.

**Principais características:**

✅ **Herança**: Estende `IRepository<NotaFiscal>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetNotaFiscalListForDropDown()`

**Descrição**: Retorna lista de notas fiscais para DropDownList

---

### `Update(NotaFiscal notaFiscal)`

**Descrição**: Atualiza nota fiscal com lógica específica
