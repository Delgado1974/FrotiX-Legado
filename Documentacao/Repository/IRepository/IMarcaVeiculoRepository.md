## Visão Geral

A interface `IMarcaVeiculoRepository` define o contrato específico para operações de repositório de marcas de veículos.

**Principais características:**

✅ **Herança**: Estende `IRepository<MarcaVeiculo>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetMarcaVeiculoListForDropDown()`

**Descrição**: Retorna lista de marcas de veículos para DropDownList

---

### `Update(MarcaVeiculo marcaVeiculo)`

**Descrição**: Atualiza marca de veículo com lógica específica
