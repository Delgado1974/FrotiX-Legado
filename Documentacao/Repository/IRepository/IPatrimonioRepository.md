# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `IPatrimonioRepository` define o contrato específico para operações de repositório de patrimônios.

**Principais características:**

✅ **Herança**: Estende `IRepository<Patrimonio>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetPatrimonioListForDropDown()`

**Descrição**: Retorna lista de patrimônios para DropDownList

---

### `Update(Patrimonio patrimonio)`

**Descrição**: Atualiza patrimônio com lógica específica
