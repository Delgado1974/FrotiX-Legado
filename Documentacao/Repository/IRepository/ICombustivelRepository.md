# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `ICombustivelRepository` define o contrato específico para operações de repositório de combustíveis.

**Principais características:**

✅ **Herança**: Estende `IRepository<Combustivel>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetCombustivelListForDropDown()`

**Descrição**: Retorna lista de combustíveis para DropDownList

---

### `Update(Combustivel combustivel)`

**Descrição**: Atualiza combustível com lógica específica
