# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `IFornecedorRepository` define o contrato específico para operações de repositório de fornecedores.

**Principais características:**

✅ **Herança**: Estende `IRepository<Fornecedor>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetFornecedorListForDropDown()`

**Descrição**: Retorna lista de fornecedores para DropDownList

---

### `Update(Fornecedor fornecedor)`

**Descrição**: Atualiza fornecedor com lógica específica
