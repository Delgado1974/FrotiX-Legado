# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `IUnidadeRepository` define o contrato específico para operações de repositório de unidades.

**Principais características:**

✅ **Herança**: Estende `IRepository<Unidade>`  
✅ **Dropdown**: Método para lista de seleção

---

## Métodos Específicos

### `GetUnidadeListForDropDown()`

**Descrição**: Retorna lista de unidades para DropDownList

---

### `Update(Unidade unidade)`

**Descrição**: Atualiza unidade com lógica específica
