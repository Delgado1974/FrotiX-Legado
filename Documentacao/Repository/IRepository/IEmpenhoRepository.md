# PARTE 1: DOCUMENTAÇÃO DA FUNCIONALIDADE

## Visão Geral

A interface `IEmpenhoRepository` define o contrato específico para operações de repositório de empenhos.

**Principais características:**

✅ **Herança**: Estende `IRepository<Empenho>`  
✅ **Dropdown**: Método para lista de seleção com JOIN

---

## Métodos Específicos

### `GetEmpenhoListForDropDown()`

**Descrição**: Retorna lista de empenhos para DropDownList com informações do contrato

---

### `Update(Empenho empenho)`

**Descrição**: Atualiza empenho com lógica específica
